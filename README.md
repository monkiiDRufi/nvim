# Neovim Configuration

Pluginless setup, exploring the native tools of Neovim.

## Features

- **LSP Support**: Python (basedpyright), Lua (lua-language-server)
- **Native Completion**: nvim-cmp integration with LSP
- **Auto-formatting**: On-save formatting via LSP
- **Git Integration**: Git blame for current line
- **Terminal Keybindings**: Easy navigation between terminal and splits
- **Native Search**: `:find` for file search, `:vimgrep` for text search


## External Tools

- `basedpyright-langserver` — Python static type checker & LSP
- `lua-language-server` — Lua LSP
- `git` — Version control (blame integration)
